## üì¶ Setup & Development

1. **Clone the repo:**
   ```sh
   git clone https://github.com/NITHINKR06/emp.git
   cd emp
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start development server:**
   ```sh
   npm run dev
   ```

4. **Build for production:**
   ```sh
   npm run build
   ```

5. **Preview production build:**
   ```sh
   npm run preview
   ```
